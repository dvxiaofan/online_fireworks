# Online Fireworks 开发方案

## 目标

实现一个基于 Vue 3 + Vite 的前端烟花页面。用户点击页面后，系统会从窗口底部发射一枚烟花弹，烟花弹经过明显的升空过程后，在随机高度绽放。每次绽放的烟花类型、颜色、粒子规模、爆炸形态都有随机变化，整体效果尽量接近真实烟花。

## 核心体验

- 页面全屏展示夜空和烟花效果。
- 用户点击任意位置时触发一次烟花发射。
- 烟花不是在点击点直接爆炸，而是从页面底部升空。
- 烟花的水平目标位置接近用户点击位置。
- 绽放高度随机，但限制在合理视觉区间，避免太低或超出屏幕。
- 每一发烟花随机选择不同花型。
- 支持连续点击，多个烟花可以同时升空和绽放。
- 支持桌面鼠标点击和移动端触摸。

## 推荐技术方案

使用 Vue 3 + Vite 搭建项目，烟花动画使用原生 Canvas 2D 实现。

不优先使用成品烟花库，原因是本项目的关键交互是“点击后从底部升空，到随机高度再随机花型绽放”。现成库通常更偏向背景烟花或点击点直接爆炸，自定义发射轨迹和爆炸逻辑不如自写 Canvas 粒子系统灵活。

## 项目结构建议

```text
online_fireworks/
  package.json
  index.html
  vite.config.ts
  tsconfig.json
  src/
    main.ts
    App.vue
    style.css
    components/
      FireworksCanvas.vue
    composables/
      useFireworks.ts
    types/

```

## 主要模块

### App.vue

负责页面外壳，挂载全屏烟花组件。

### FireworksCanvas.vue

负责创建全屏 Canvas，监听窗口尺寸变化和 pointer 事件，并调用烟花引擎。

### useFireworks.ts

烟花核心引擎，负责：

- 管理 Canvas 渲染上下文。
- 管理动画循环。
- 管理火箭、爆炸粒子、尾焰、余烬。
- 根据点击位置创建新的烟花。
- 随机选择烟花类型。
- 根据设备像素比适配高清屏。

## 烟花生命周期

### 1. 发射

点击页面时创建一枚 Rocket。

Rocket 初始位置：

- x：窗口底部中心附近，或点击位置附近带少量随机偏移。
- y：窗口底部之外一点，形成从屏幕外升起的感觉。

Rocket 目标位置：

- targetX：接近点击位置 x，并加入轻微随机偏移。
- targetY：随机高度，建议在窗口高度的 18% 到 60% 之间。

运动方式：

- 使用速度向量向目标点移动。
- 加入轻微横向漂移。
- 加入逐帧尾焰粒子。
- 到达目标点附近或速度衰减后触发爆炸。

### 2. 绽放

Rocket 到达目标高度后，按随机花型创建一组 Particle。

Particle 具有：

- 位置
- 速度
- 半径
- 颜色
- 透明度
- 生命周期
- 重力
- 空气阻力
- 闪烁概率
- 发光强度

### 3. 消散

粒子逐渐受重力下落、速度减弱、透明度降低，最终从粒子数组中移除。

## 随机烟花类型

第一版建议实现以下类型：

### Chrysanthemum 菊花型

最经典的圆形爆炸。粒子向四周均匀扩散，颜色明亮，外圈较饱满。

特点：

- 粒子数量多。
- 爆炸半径大。
- 适合作为默认花型。

### Peony 牡丹型

圆形但更厚重，粒子速度较慢，亮度更强，视觉上更饱满。

特点：

- 粒子更密。
- 半径略大。
- 衰减较慢。

### Ring 环形

粒子集中在一个圆环上，中心较空。

特点：

- 粒子角度均匀。
- 速度差异小。
- 视觉形态清晰。

### Willow 柳树型

粒子爆开后明显下垂，像柳条一样落下。

特点：

- 重力更强。
- 粒子寿命更长。
- 下坠拖尾明显。

### Palm 棕榈型

少量粗亮粒子向外展开，形成棕榈叶形态。

特点：

- 粒子数量较少。
- 单粒子更亮更粗。
- 方向分布有主枝感。

### Crackle 闪烁爆裂型

主爆炸后有短促闪烁和小火星。

特点：

- 粒子透明度带随机闪烁。
- 后期产生少量二级火星。
- 适合制造真实感。

## 真实感关键点

### 发光混合

Canvas 使用：

```ts
ctx.globalCompositeOperation = 'lighter'
```

这样不同粒子的光会叠加，接近真实烟花的亮部效果。

### 拖尾残影

每帧不是完全清空画布，而是覆盖一层半透明深色：

```ts
ctx.fillStyle = 'rgba(2, 5, 12, 0.22)'
ctx.fillRect(0, 0, width, height)
```

这样可以保留短暂轨迹，形成光迹。

### 物理参数

建议参数范围：

- gravity：0.035 到 0.09
- friction：0.965 到 0.99
- particle count：60 到 180
- particle life：60 到 150 帧
- rocket speed：8 到 13
- rocket trail count：2 到 5

### 高分屏适配

Canvas 实际尺寸需要乘以 `window.devicePixelRatio`，CSS 尺寸保持 100vw / 100vh。

### 性能控制

- 使用单个 requestAnimationFrame 循环。
- 粒子生命结束后及时移除。
- 限制同时存在的粒子总数，例如 2500 到 4000。
- 连续点击过快时仍允许响应，但可对每帧创建数量做上限保护。

## 页面视觉

背景建议是纯 Canvas 渲染的深夜天空：

- 顶部深蓝接近黑色。
- 底部略微偏暖或偏紫。
- 可以加入极少量星点，但不要喧宾夺主。
- 页面上不放说明卡片，不影响沉浸感。

可以在角落保留非常轻的标题或状态文字，但默认建议隐藏 UI，只保留烟花。

## 交互细节

- `pointerdown` 触发发射。
- 鼠标或手指点击位置决定烟花大致目标 x。
- 目标 y 随机，而不是直接使用点击 y。
- 可以每次点击发射 1 枚，也可以小概率发射 2 到 3 枚联发。
- 可选：首次进入页面时自动发射 2 到 3 枚作为欢迎效果。

## 第一版实现范围

第一版建议完成：

- Vue3 + Vite 项目搭建。
- 全屏 Canvas 页面。
- 点击从底部升空。
- 随机爆点高度。
- 至少 5 种随机烟花花型。
- 粒子发光、拖尾、重力、摩擦、闪烁。
- 桌面和移动端响应。
- 本地运行和构建验证。

## 后续增强方向

- 增加烟花升空音效和爆炸音效。
- 增加烟雾粒子。
- 增加文字烟花或心形烟花。
- 增加节日主题配色。
- 增加自动播放模式。
- 增加参数面板，调节粒子数量、重力、亮度、背景等。

## 开发步骤

1. 初始化 Vue3 + Vite 项目。
2. 创建全屏 Canvas 组件。
3. 实现动画循环和画布 resize。
4. 实现 Rocket 升空逻辑。
5. 实现 Particle 爆炸粒子。
6. 实现随机烟花类型。
7. 加入拖尾、发光、闪烁、尾焰。
8. 做性能限制和移动端适配。
9. 运行开发服务器进行视觉验证。
10. 构建项目确认无类型和打包错误。

## 当前决策

推荐采用“自写 Canvas 粒子系统”的方案。它能更准确满足以下需求：

- 点击触发。
- 从底部升空。
- 绽放高度随机。
- 烟花类型随机。
- 效果细节可继续打磨。
