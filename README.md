# Online Fireworks

Vue 3 + Vite front-end project for an interactive realistic fireworks page.

The page will use a full-screen Canvas particle system. Clicking or tapping the page launches a firework from the bottom of the viewport, then it blooms at a randomized height with a randomized firework type.
